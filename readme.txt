== Description ==
RSS feed for Events Plugin by Arnan de Gans. 
Outputs the Events as RSS-Feed. The output of the feed description is fully customizable. 
To view the feed, use a link like http://yourblog.tld/?eventsrss.

== Installation ==
Just download and install the plugin. Configure it to your needs.
